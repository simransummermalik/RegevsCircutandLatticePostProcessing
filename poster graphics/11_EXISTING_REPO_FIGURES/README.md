# Existing figures from repository runs

These are convenient copies of figures already present elsewhere in the
repository. The filename prefix tells you how safely each can be used.

## Recommended for the final poster

- `PRIMARY_current_certificate_vs_recovery.png` is the current frozen
  certificate-gap figure. Its **left panel uses representative N=55 only**;
  its right panel aggregates paired differences over all eight held-out
  moduli. State that in the caption.

## Background or backup-slide material

- Files beginning with `BACKGROUND_superseded_` are from earlier QFT stages.
  Their analytic certificate results remain useful context, but the current
  held-out certificate-gap study superseded them as the headline result.
- `BACKGROUND_standard_parseval_identity.png` visualizes a standard
  Parseval/character-orthogonality identity. Do not call the identity novel.
- `BACKGROUND_earlier_entropy_relation_recovery.png` is exploratory context,
  not the final contribution.

## Negative-result context

- Files beginning with `NEGATIVE_` document hypotheses that did not generalize
  or did not beat their matched endpoint. They are valuable for a research
  process or backup slide, but should not be shown as positive evidence for
  base selection.

The newly generated poster-native plots elsewhere in `poster graphics/` use
larger labels, a consistent palette, and explicit evidence-level captions.
